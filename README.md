# Mission Control v2

A centralized command center dashboard for tracking career progress, project management, and personal goals.

## Features

- **Dashboard**: Real-time metrics and activity feed
- **Projects**: Kanban board for task management
- **Timeline**: Career roadmap visualization
- **Notes**: Auto-saving markdown editor
- **Mobile Responsive**: Optimized for all devices
- **Foreman Integration**: Syncs with OpenClaw project data

## Live Deployment

Deployed on Vercel: [https://mission-control-v2.vercel.app](https://mission-control-v2.vercel.app)

## Local Development

```bash
# Clone the repository
git clone https://github.com/Akira627/mission-control-v2.git

# Run local server (optional)
node server.js

# Open in browser
open http://localhost:8899
```

## Project Structure

- `mission-control-v2.html` - Main dashboard (all CSS/JS inline)
- `mobile-optimization.css` - Mobile-specific styles
- `projects-data.json` - Sample project data
- `server.js` - Local development server
- `vercel.json` - Vercel deployment configuration

## Technology Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Inter)
- Vercel for hosting
